# Collate Notes

# This application is not actively maintained anymore.

You are welcome to fork it and create your own note taking application. Licensed under MIT, so have at it.

# Quickstart

Install dependencies

```
npm install
```

Run development server

```
npm run dev
```

Build for platform

```
npm run build:mac
npm run build:linux
npm run build:win32
npm run build:win64
```
